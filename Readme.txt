Mount&Blade Module System

For getting documentation and the latest version of the module system check out:

www.taleworlds.com/mb_module_system.html

to compile this mod:

go into module_info.py and specify an output location(preferbly the location of a copy of your native mod)

run the batch script that came with this repo

make sure your txts are located in a copy of the native mod

make sure your mod is located in the module folder of the game

open the game and select the new module to load
